📖 Overview

A C#-based Hospital Management System designed to manage patient records, admissions, and reports efficiently.
This project is built for learning purposes and demonstrates core programming concepts with a structured approach.

---

✨ Features

✔️ Patient Admission & Discharge
✔️ Blood Group Management
✔️ Patient Report Generation
✔️ Simple & Clean Console Interface
✔️ Easy Data Handling

---

🛠️ Tech Stack

- 💻 Language: C#
- ⚙️ Framework: .NET
- 🧰 IDE: Visual Studio
